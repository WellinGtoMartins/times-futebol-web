## Descrição
  O principal objetivo desse projeto foi praticar as tecnologias usadas no desenvolvimento Java para WEB de forma prática desenvolvendo um projeto que trata do gerenciamento de times de futebol, permitindo adicionar, editar e excluir times da sua lista.

## Ambiente de Desenvolvimento
- JDK 17
- IDE Intellij
- Wildfly 31.0.0
- MySQL

## Tecnologias
- Maven
- JSF (JavaSever Faces)
- PrimeFaces
- JTA (Java Transaction API)
- JPA (Java Persistence API)
- Hibernate
- EJB (Enterprise JavaBeans)
